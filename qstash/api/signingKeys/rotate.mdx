---
title: "Rotate Signing Keys"
description: "Rotate your signing keys"
api: "POST https://qstash.upstash.io/v2/keys/rotate"
authMethod: "bearer"
---

## Response

<ResponseField name="current" type="string" required>
  Your current signing key.
</ResponseField>
<ResponseField name="next" type="string" required>
  The next signing key.
</ResponseField>

<RequestExample>

```sh curl
curl https://qstash.upstash.io/v2/keys/rotate \
  -H "Authorization: Bearer <token>"
```

```javascript Node
fetch('https://qstash.upstash.io/v2/keys/rotate', {
  headers: {

  }
});
```
